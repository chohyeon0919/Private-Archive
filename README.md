# 🎬 Private Culture Archiver (문화생활 기록 웹 앱)
타인의 시선 없이 오롯이 나만의 솔직한 감상을 안전하게 기록하고 보관하는 비공개 콘텐츠 아카이빙 웹 앱

---

## 📌 1. 프로젝트 설명
- **기획 의도:** 왓챠피디아 등 기존 플랫폼은 익명이라고 하더라도 리뷰가 타인에게 강제로 공개되어 내향적인 사용자들이 솔직한 후기를 남기기 어려울 수 있습니다. 타인의 시선을 신경 쓰지 않고 자신만의 감상을 독립적으로 보관할 수 있는 공간을 만들고자 프로젝트를 시작했습니다.
- **대상 콘텐츠:** 책, 영화, 드라마
- **핵심 특징:** 외부 서버나 DB 연동 없이 브라우저의 로컬 저장소를 활용하여 후기가 외부로 유출되지 않도록 보안성과 사적 영역을 확보했습니다.

---

## 🚀 2. 사용 방법
웹 앱에 접속한 후, 아래 순서에 따라 나만의 프라이빗한 문화생활을 기록하고 관리할 수 있습니다.

1. **카테고리 선택**: 화면 상단에서 기록하고자 하는 콘텐츠의 종류 [책], [영화], [드라마] 중 하나를 선택합니다.
2. **기본 정보 입력**: 
   - 기록할 작품의 **제목**을 입력합니다 (미입력 시 "콘텐츠 제목을 입력해 주세요."라는 빨간색 경고 문구가 표시됩니다).
   - 작품의 **포스터 또는 표지 이미지 URL 링크**를 입력합니다 (주소를 비워두면 자동으로 기본 대체 아이콘이 지정됩니다).
3. **별점 매기기**: 5개의 별 아이콘을 클릭하여 최소 0점부터 최대 5점까지 작품의 별점을 선택합니다.
4. **후기 작성**: 하단의 텍스트 창에 나만의 솔직한 감상 평을 장문으로 자유롭게 작성합니다 (글이 길어지면 내부에 스크롤바가 생성됩니다).
5. **기록하기**: 모든 입력을 마친 후 **[기록하기]** 버튼을 누르면, 현재 컴퓨터의 시스템 날짜가 자동으로 기록되며 브라우저 내부 저장소(LocalStorage)에 안전하게 보관됩니다.
6. **대시보드 확인 및 상세 보기**: 저장과 동시에 메인 화면의 바둑판식 배열에 나만의 카드가 실시간으로 추가되며, 카드를 클릭하면 뒷배경이 흐려지면서 하얀색 상세 보기 팝업창을 통해 작성한 글을 오롯이 확인할 수 있습니다.
7. **기록 삭제**: 수정이나 삭제가 필요할 경우, 상세 보기 창 내부의 [삭제] 버튼을 누르고 확인 창에서 승인하면 대시보드와 저장소에서 즉시 실시간으로 동기화되어 삭제됩니다.

## ⚠️ 4. 한계점 및 향후 개선 사항
- **도메인 배포 미적용:** 현재 로컬 환경에서 `index.html` 파일을 직접 실행하는 형태로 구동됩니다. 향후 GitHub Pages 등을 활용한 웹 호스팅 배포가 필요합니다.
- **로컬 스토리지의 한계:** 브라우저 자체의 `LocalStorage`를 이용하기 때문에, 사용자가 브라우저 쿠키 및 캐시를 강제로 모두 삭제하거나 다른 컴퓨터/기기에서 접속할 경우 기존에 작성했던 기록을 이어볼 수 없습니다.
- **이미지 직접 업로드 기능:** 현재는 웹에 등록된 이미지의 URL 주소를 직접 복사해서 입력해야만 이미지가 출력되며, 내 컴퓨터에 있는 포스터 사진 파일을 직접 선택해서 업로드하는 기능은 아직 지원하지 않습니다.
